3.  The "ietf-template" YANG Module

3.1.  Data Model Overview

   The following tree diagram [RFC8340] illustrates the "ietf-template"
   module:

   module: ietf-template
     +--rw templates* [name]
        +--rw name    string
        +--rw data?   <anydata>

3.2.  YANG Module

   <CODE BEGINS> file "ietf-template@2024-08-27.yang"
   module ietf-template {
     yang-version 1.1;
     namespace "urn:ietf:params:xml:ns:yang:ietf-template";
     prefix template;

     organization
       "IETF NETMOD (Network Modeling) Working Group";
     contact
       "WG Web:   <https://datatracker.ietf.org/wg/netmod/>
        WG List:  <mailto:netmod@ietf.org>

        Author: Qiufang Ma
                <mailto:maqiufang1@huawei.com>";
     description
       "This module defines a template.

        Copyright (c) 2024 IETF Trust and the persons identified
        as authors of the code. All rights reserved.

        Redistribution and use in source and binary forms, with
        or without modification, is permitted pursuant to, and
        subject to the license terms contained in, the Revised
        BSD License set forth in Section 4.c of the IETF Trust's
        Legal Provisions Relating to IETF Documents
        (https://trustee.ietf.org/license-info).

        This version of this YANG module is part of RFC XXXX
        (https://www.rfc-editor.org/info/rfcXXXX); see the RFC
        itself for full legal notices.

        The key words 'MUST', 'MUST NOT', 'REQUIRED', 'SHALL',
        'SHALL NOT', 'SHOULD', 'SHOULD NOT', 'RECOMMENDED',
        'NOT RECOMMENDED', 'MAY', and 'OPTIONAL' in this document
        are to be interpreted as described in BCP 14 (RFC 2119)
        (RFC 8174) when, and only when, they appear in all
        capitals, as shown here.";

      list templates {
        key name;
        leaf name {
          type string;
          description
            "The name of the template that uniquely identifies a
             template.";
        }
        anydata data {
          description
            "inline template content.";
        }
      }
   }
